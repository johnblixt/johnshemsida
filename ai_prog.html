Dessa ska laddas upp på github och i index.html finnas länkat till cv.html, gymnasiearbete.html och games.html. Ge mig ett första utkast."

Kopiera texten ovan och klistra in i Gemini.

Guide: Publicera (GitHub & Netlify)

Del 1: Skapa en Repository på GitHub

  1. Logga in: Gå till github.com.
  2. Ny Repository: Klicka på + i hörnet → New repository.
  3. Namnge: Ge projektet ett namn (t.ex. min-ai-sida).
  4. Inställningar: Se till att den är Public. Klicka på Create repository.
  5. Skapa startsidan: Klicka på länken "creating a new file" mitt på sidan.
  6. index.html: Döp filen till index.html. Kopiera in koden från Gemini för startsidan i textfältet och välj Commit changes.
  7. Skapa fler filer: Klicka på Add file → Create new file för att skapa cv.html, gymnasiearbete.html och games.html.
  8. Fyll i innehåll: Kopiera in koden till respektive fil (gymnasiearbete, CV och spel) som du har fått från Gemini och klicka på Commit changes för varje fil.

Del 2: Koppla till Netlify

  1. Logga in: Gå till netlify.com (Logga in med GitHub).
  2. Ny sajt: Add new site → Import from an existing project.
  3. Välj GitHub: Auktorisera och sök efter din repository.
  4. Deploy: Lämna alla inställningar tomma och klicka på Deploy.

Viktigt: Din huvudfil måste heta exakt index.html för att hemsidan ska fungera!
